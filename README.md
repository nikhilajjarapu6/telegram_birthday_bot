# Telegram Birthday Bot

An automated Telegram bot that sends birthday wishes to your subscribers.

## What it does

Simple bot that:
- Collects subscriber birthdays
- Sends automatic birthday wishes on the right day
- Stores data in JSON format

## Quick Start

1. Get bot token from @BotFather on Telegram
2. Configure your bot with the token
3. Run: `mvn spring-boot:run`
4. Start using the bot!

## Commands

- `/start` - Start the bot
- `/subscribe` - Add your birthday
- `/unsubscribe` - Remove from list
